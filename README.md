# Melody Documentation

This is the documentation site for Melody - a music streaming platform.

## About

Melody helps you discover more and listen better. An intelligent music streaming service with personalized recommendations, social features, and artist tools.

## How to Use

1. Visit the [website](https://melody2c2025.github.io/)
2. Navigate through the sections using the buttons in the header
3. Read the guides and technical documentation

## Local Development

This site is built with Jekyll and uses the Cayman theme.

To run locally:
```bash
bundle install
bundle exec jekyll serve
```

## Adding Content

- Edit `index.md` for the About page
- Edit `manuals.md` for user guides
- Edit `tech_doc.md` for technical documentation
- Add images to the `assets/` folder

## Deployment

This site is designed to be deployed on GitHub Pages.

Push to the main branch and GitHub will automatically build and deploy the site.
